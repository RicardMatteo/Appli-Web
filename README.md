# Appli-Web
Projet d'application web réalisé au S8 de la formation SN de l'n7

## Installation

### Java

Utiliser Jboss et mettre le WAR dans standalone/deployments 

Backend situé sur `localhost:8080`

### React

Installer nodeJS.

```bash
cd ade-en-mieux
npm install
npm start
```

Application react située sur `localhost:3000`
